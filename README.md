Installation Guide:

NOTE: if you dont want to setup the database manually, run any of the unittests and it will run a rebuild script for the database.
SETUP an environmental variable named pgconn with a database connection.

1. cd frontend

2. npm install

3. ng build

4. cd ..

5. cd API

6. dotnet run
